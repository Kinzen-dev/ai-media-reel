ATELIER / BLIND WIPE
SVG MASK REVEAL
04 PATTERNS

Works behind blinds, opened by the scroll.

Each piece is sealed under a black SVG mask. Reach it and the curtain runs one of four wipes - horizontal blinds, a shuffled grid, vertical blinds, a column wave - the vermilion seam flashing along every closing edge. Native masks, no libraries.

Open the blinds.
Engine: one SVG <mask> per work; black panels are <rect>s animated by CSS width/height transitions, so reveal = panel retract.
Mechanic: IntersectionObserver arms each stage; the first work is held mid-wipe at load so the gallery thumbnail already shows the move.
Quiet: prefers-reduced-motion collapses all four to an instant opacity reveal.
