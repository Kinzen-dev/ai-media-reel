Living Grid / Container Queries

One card, five layouts. You hold the density.

Drag the handle to set the column count from 1 to 5. Every card reshapes its own internal layout with CSS @container queries - no script decides what a card looks like. Editorial spread, then stacked card, compact thumbnail, image tile, dense sliver.

Pinch / rotate your phone - the grid is responsive here. The drag handle is a desktop control.
Mechanic: column count sets only --cols; each card self-governs via @container card (min-width: ...). Fallback: @supports not (container-type) -> standard media-query grid.
