Data as architecture

Your metrics, built as a voxel skyline.

Every tower is one data point. Height encodes the value, color ramps cool-teal at the low end to warm-coral at the peak. Pure CSS 3D transforms, no WebGL, no canvas. Swap the DATA array and the city rebuilds itself.

Encoding

Height + color, two channels

Tower height maps the raw value and the same value drives a teal-to-coral ramp, so outliers spike and glow without a single axis label.

Pure CSS 3D

Three faces per voxel

Each building is a lit top plus two shaded side faces from CSS transforms. No WebGL, no canvas, no animation library - it stays well above 50fps.

Swap-ready

One DATA array

Edit the labeled array at the top of the file. Set a value, optionally a name and unit, and the grid, legend, and peak readout all recompute.
