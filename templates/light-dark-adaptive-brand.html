No theme classes. No duplicated variables.

The old way ships two color blocks and a class toggle that can drift out of sync. The native way ships one block. The browser picks the side from color-scheme.

  • One source of truth per color, impossible to desync.
  • OS preference honored on first paint, no flash.
  • Nested color-scheme lets one component pin its own side.
tokens.css
/* declare once, resolve twice */
:root {
  color-scheme: light dark;

  --bg:    light-dark(#f4f1ea, #0c0b0d);
  --ink:   light-dark(#15130f, #f4f1ea);
  --line:  light-dark(#d8d2c4, #2a262b);
  --accent: #ff5a3c; /* constant */
}

/* the toggle does only this */
html[data-scheme="dark"] {
  color-scheme: dark;
}
