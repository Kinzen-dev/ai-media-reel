01 / 04 ●MOSAIC

Layout is not decoration. It is an argument about what matters, made over and over, in real time.

Paradigm

Morphing grid. One continuous set of cells animates between four formations driven entirely by scroll position, no animation library.

Engine

Layout solvers return per-cell geometry. Adjacent layouts are interpolated with eased transforms written each frame inside one rAF loop.

Media

Posters by default. Only the focused cell decodes and plays its loop, so simultaneous video never tanks frame rate.
